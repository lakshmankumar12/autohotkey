## Alphabetical order

```
a - WINDWS - alt-esc
b - HOTKEY - Adobe
c - FRKEY  - ctrl-c (dont change - this help help to copy as you expect ctrl-c)
d - NVGTN  - Page-Down
e - NVGTN  - load clipboard file
f - NVGTN  - Page-UP
g - MOUSE  - main Mouse click
h - NVGTN  - Left
i - MOUSE  - Mouse-move up - fast
j - NVGTN  - Down
k - NVGTN  - Up
l - NVGTN  - Right
m - MOUSE  - Mouse-move down - slow
n - MOUSE  - Mouse-move left - slow
o - MOUSE  - Mouse-move right - fast
p - FRKEY  - Ctrl-p
q - FRKEY  - Alt-q  (to aid the quickey extension of chrome)
r - FRKEY  - Apps key
s - HOTKEY - Activate slack
t - FRKEY  - ctrl-t
u - MOUSE  - Mouse-move down - fast
v - FRKEY  - Same as ] - Shift Ins (dont change - same as paste in windows)
w - FRKEY  - ctrl-w
x - FRKEY  - Del
y - MOUSE  - Mouse-move left fast
z - WINDWS - ctrl-alt-tab

1 - HOTKEY - Work-Putty
2 - HOTKEY - local-virt-box putty
3 - HOTKEY - Visual Studio Code
4 - FRKEY  - End
5 - WINDWS - Ask and switch window
6 - MOUSE  - Mouse Highlight
7 - FRKEY  - ctrl-u
8 - WINDWS - Right click
9 - FRKEY  - scroll lock
0 - FRKEY  - Home

[ - FRKEY  - control shift tab
] - FRKEY  - control tab

F1 - none
F2 - none
F3 - none
F4 - media pause
F5 - vol-down
F6 - vol-up
F7 - browser-back
F8 - browser-fwd
F9 - work-profile chrome
F10 - personal-profile chrome
F11 - Activate Youtube Music
F12 - win Title
```
